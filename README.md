# AstrBot 对话互通桥 (Conversation Bridge)

本插件为 AstrBot 提供了一个跨会话的消息处理能力，允许管理员在不同的私聊或群聊窗口之间进行消息代发与记录查询。

## 核心特性

- **跨对话代发**：管理员可以命令机器人向指定的 QQ 号或群号发送消息，实现远程操控。
- **记录互通**：支持跨窗口调取指定会话的历史聊天记录（需配合底层权限）。
- **权限安全锁**：通过 `admin_qqs` 配置项严格限制功能使用权限，确保隐私安全。
- **高兼容性**：支持私聊与群聊的双向互通。

## 安装方法

1. 在 AstrBot 插件管理中搜索 `conversation_bridge` 或通过本仓库 URL 安装。
2. 重启 AstrBot。

## 配置项说明

| 配置项 | 说明 | 示例 |
| :--- | :--- | :--- |
| `admin_qqs` | 允许使用桥接功能的管理员 QQ 列表（半角逗号分隔）。 | `1591793025,123456` |
